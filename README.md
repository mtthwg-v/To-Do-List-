# To-Do-List-
To-Do List CRUD Application using JavaScript and LocalStorage
This project is a simple To-Do List application that demonstrates CRUD operations using JavaScript and LocalStorage.

Features
- Create new tasks
- Read and display tasks
- Update tasks by marking them as completed
- Delete tasks

 Technologies Used
- HTML
- CSS
- JavaScript
- Browser LocalStorage

 How to Run
1. Download or clone the repository
2. Open `index.html` in a web browser

This project was created as a final activity for Computer Fundamentals and Programming.
